## 风险与应对

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| LangChain 版本变更 | API 不兼容 | 锁定版本 0.3.x |
| LLM API 不稳定 | Agent 无法运行 | 添加备用模型切换 |
| 向量检索效果差 | RAG 效果不佳 | 优化 chunk size |
| 可视化性能 | 刷新卡顿 | 使用 WebSocket 增量更新 |
